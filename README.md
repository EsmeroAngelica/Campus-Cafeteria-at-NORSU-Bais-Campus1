 Instructions: How to Run the Project
1. Open the Home Page
2. Choose how you want to continue whether a customer or admin.
3. Customer choose signup or login.
4. After signing up, login using your email and password. Browse the menu, add items to cart, and place your order.
5. Admin choose signup or login.
6. Manage Orders, View Payments, Inventory Logs, Manage Menu, Add, edit, or delete menu items in the Manage Menu section.
7. Logout
